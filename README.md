# Telegram Message Sender

Простой скрипт на Python для отправки сообщений через личный Telegram аккаунт с использованием Telethon.

## Возможности

- Отправка сообщений по списку username
- Поддержка нескольких вариантов текста
- Случайный выбор сообщения
- Задержки между отправками
- Обработка FloodWaitError
- Удаление дубликатов username
- Ограничение количества сообщений за запуск
- Логирование ошибок

---

## Использование

Скрипт читает файл `users.txt`, находит все Telegram username и отправляет каждому случайное сообщение из списка `MESSAGES`.

Пример username:

```txt
@username1
@username2
@username3
